## рҹҺҜ What is CrowdSense?

CrowdSense is a **real-time venue intelligence Progressive Web App (PWA)** that transforms the experience of attending any large event вҖ” sports matches, concerts, F1 races, comedy shows, and more.

The app provides fans with:
- **Live crowd heatmaps** so you always know where to go
- **AI-powered navigation** to avoid peak queues
- **In-seat food ordering** with live wait time tracking
- **Smart gate recommendations** to minimize entry time
- **Gamification** вҖ” earn points, unlock badges, redeem rewards

> Built with a **Kinetic HUD** design philosophy вҖ” dark, high-contrast, data-dense, and alive with real-time feedback.

## рҹҢҹ Features

### рҹ“Қ Venue Map
- Generic SVG map that works for **any event type** (Cricket, Football, F1, Concerts, Comedy)
- **Heatmap mode** вҖ” color-coded crowd density per zone
- **2D Plan mode** вҖ” structural layout view
- Tap any zone to see occupancy %, capacity, wait time, trend
- Gate status indicators (G1вҖ“G4) with open/closed status
- Your seat location pinned on the map

### рҹҚ” Food & Ordering
- Browse menu items from 5 food counters
- Filter by cuisine, veg/non-veg, category
- Live **wait time indicators** per counter
- Add to cart вҶ’ place order вҶ’ real-time status tracking (Placed вҶ’ Preparing вҶ’ On the Way вҶ’ Delivered)
- Floating persistent cart widget

### рҹҺ« Wallet
- Live match ticket with QR code (scan at gate)
- **6 upcoming events** across:
  - рҹҸҸ Cricket (IPL)
  - рҹҺө Concert (Coldplay)
  - рҹҸҺпёҸ F1 (Bahrain GP)
  - вҡҪ Football (ISL Final)
  - рҹҺӨ Comedy (Stand-up)
- View/Buy Ticket for each event
- Season attendance heatmap

### рҹ“Ҡ My Stats
- Activity timeline for the day
- **Polished route map** вҖ” animated Bezier curved paths with direction arrows
- AI impact metrics (time saved, queues avoided, steps saved)
- Achievements & badges
- Your stats vs. average attendee

### рҹӣЎпёҸ Admin Dashboard
- Real-time KPI cards (crowd count, high-risk zones, wait times, active alerts)
- **Interactive trend chart** вҖ” hover for tooltip, click to pin data points with glow
- Live alert feed
- Broadcast platform-wide alerts
- Event simulation buttons (Goal, Halftime, Match End, Emergency Drill)
- Service counter wait time management

### вҡҷпёҸ Settings
- All toggles **fully wired** to user preferences:
  - Live Updates, Food Alerts, Haptic Feedback, AR Features
  - Dark/Light mode, Compact View, Data Sharing
- **Haptic feedback** via `navigator.vibrate()` on supported devices
- Firebase `signOut()` on logout

### рҹ‘Ө Profile
- Inline name editing (no modal)
- XP bar with tier progression (Bronze вҶ’ Silver вҶ’ Gold вҶ’ Platinum вҶ’ Legend)
- Badges earned display

## рҹҡҖ Getting Started

### Prerequisites
- Node.js 20+
- npm 10+

### 1. Clone & Install

```bash
git clone https://github.com/risshiisshh/CrowdSense.git
cd CrowdSense
npm install
```

### 2. Run in Demo Mode (no Firebase needed)

```bash
npm run dev
# вҶ’ http://localhost:5173
```

The app runs with rich mock data вҖ” perfect for development.

### 3. Enable Firebase Backend (optional)

Create a `.env` file:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=123456789
VITE_FIREBASE_APP_ID=1:123456789:web:abcdef
VITE_USE_FIREBASE=true
```

Then restart the dev server. See `.env.example` for a template.

## вҳҒпёҸ Deploy to Google Cloud Run

```bash
# One-time setup
gcloud auth login
gcloud config set project YOUR_PROJECT_ID
gcloud services enable cloudbuild.googleapis.com run.googleapis.com artifactregistry.googleapis.com

# Deploy
gcloud run deploy crowdsense \
  --source . \
  --region asia-south1 \
  --platform managed \
  --allow-unauthenticated \
  --port 8080 \
  --memory 256Mi \
  --min-instances 0 \
  --max-instances 3 \
  --set-build-env-vars "VITE_FIREBASE_API_KEY=...,VITE_USE_FIREBASE=true"
```

After deployment, add your Cloud Run URL to **Firebase Console вҶ’ Authentication вҶ’ Authorized Domains**.
